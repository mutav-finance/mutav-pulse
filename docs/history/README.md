# Design history (superseded)

The plans and specs in this folder are the **build-evolution record** of Mutav
Pulse, from the first monolith through the modular refactor. They are kept for
context but are **superseded** — they predate the **2026-06-27 fiança redesign**
and describe earlier designs (single-leg coverage, `premium`/`collect_premium`
naming, a pre-modular `reserve` monolith, "TGA" branding). They do **not** reflect
the shipped contracts.

**For the current design, see:**
- [`docs/specs/`](../specs/) — the current design specs (two-leg fiança, BRL-native reserve)
- [`docs/whitepaper.md`](../whitepaper.md) — the economic model
- [`docs/vault-method-surface.md`](../vault-method-surface.md) — the live vault method surface
- the contracts themselves (`contracts/`)

```
plans/   one implementation plan per phase (2026-06-20 → 06-23)
specs/   one design spec per phase   (2026-06-20 → 06-23)
```
